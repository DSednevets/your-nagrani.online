## 🎨 ЦВЕТОВАЯ ПАЛИТРА

### Основные цвета
```
#FFFFFF  - Фон (белый, главный)
#000000  - Черный (текст, акценты, кнопки)
#1A1A1A  - Очень темный серый (основной текст)
#666666  - Серый (вторичный текст, комментарии)
#E0E0E0  - Светло-серый (границы, разделители)
#F5F5F5  - Очень светлый серый (фон блоков)
```

### Функциональные цвета
```
#10B981  - Зеленый (успех, активное состояние)
#EF4444  - Красный (ошибка, предупреждение)
#3B82F6  - Синий (ссылки, информация) - если нужно
```

### Разница от nagrani.online
| Элемент | nagrani.online | your-nagrani.online |
|---------|----------------|---------------------|
| Фон | #000000 (черный) | #FFFFFF (белый) |
| Текст | #FFFFFF (белый) | #1A1A1A (темный) |
| Акценты | #FFFFFF | #000000 |
| Границы | Светлые серые | Светло-серые |

**Суть:** Инвертированная цветовая схема, но логотип, типография и структура остаются узнаваемы

## 🔤 ТИПОГРАФИЯ

### Шрифт
**Основной шрифт:** Inter (как на nagrani.online)

```css
/* В tailwind.config.ts */
fontFamily: {
  sans: ['Inter', 'ui-sans-serif', 'system-ui', 'sans-serif'],
}
```

### Размеры и иерархия

| Элемент | Размер | Вес | Примеры |
|---------|--------|-----|---------|
| H1 (Заголовок страницы) | 32px | 700 | "YOUR-NAGRANI", "Исследование" |
| H2 (Подзаголовок) | 24px | 600 | "Твоя трансформация", "Как это работает" |
| H3 (Подзаголовок) | 18px | 600 | "Блок ИССЛЕДОВАНИЕ", "Ответ ассистента" |
| Body Large | 16px | 400 | Основной текст, описания |
| Body Normal | 14px | 400 | Сообщения в чате, текст блоков |
| Body Small | 12px | 400 | Дата, время, вспомогательный текст |
| Label | 12px | 500 | Кнопки, метки |

### Line Height (расстояния между строками)
```
H1, H2: 1.2
H3: 1.3
Body: 1.5
Labels: 1.4
```

### Letter Spacing (межбуквенный интервал)
```
Нормальный текст: 0
Заголовки: -0.5px (чуть компактнее)
Labels на кнопках: 0.5px
```

### Примеры типографии в коде (Tailwind)

```html
<!-- H1 -->
<h1 class="text-4xl font-bold leading-tight text-black">YOUR-NAGRANI</h1>

<!-- H2 -->
<h2 class="text-2xl font-semibold leading-snug text-black">Исследование личности</h2>

<!-- H3 -->
<h3 class="text-lg font-semibold text-black">Как это работает</h3>

<!-- Body Large -->
<p class="text-base font-normal leading-relaxed text-gray-900">
  Основной текст...
</p>

<!-- Body Normal -->
<p class="text-sm font-normal leading-relaxed text-gray-800">
  Текст сообщения...
</p>

<!-- Body Small / Secondary -->
<p class="text-xs font-normal text-gray-600">
  Вспомогательный текст
</p>

<!-- Label / Button -->
<button class="text-sm font-medium tracking-wide">
  Отправить
</button>
```

## 📐 СПЕЙСИНГ И ОТСТУПЫ

### Основная сетка (8px система)

```
xs: 4px   (редко используется)
sm: 8px   (малые отступы)
md: 16px  (стандартные отступы)
lg: 24px  (большие отступы)
xl: 32px  (секции)
2xl: 64px (контейнеры)
```

### В Tailwind классах

```
p-1 = 4px
p-2 = 8px
p-3 = 12px
p-4 = 16px
p-6 = 24px
p-8 = 32px
gap-4 = 16px (между элементами)
```

### Примеры макетов с спейсингом

```html
<!-- Контейнер страницы -->
<div class="max-w-6xl mx-auto px-4 sm:px-6 lg:px-8">
  <!-- p-4 на мобильных, p-6 на планшетах, p-8 на десктопе -->
</div>

<!-- Между секциями -->
<section class="py-8 sm:py-12 lg:py-16">
  <!-- 32px сверху/снизу на мобильных, 48px на десктопе -->
</section>

<!-- Карточка -->
<div class="p-4 sm:p-6 rounded-lg border border-gray-200">
  <!-- Инбоксинг паддинг -->
</div>
```

## 🎯 RESPONSIVE DESIGN

### Breakpoints (как в Tailwind)

| Device | Breakpoint | Пример класса |
|--------|-----------|---------------|
| Mobile | 0px | `text-base` |
| Tablet | 640px (sm:) | `sm:text-lg` |
| Desktop | 1024px (lg:) | `lg:text-xl` |
| Wide | 1280px (xl:) | `xl:text-2xl` |

### Примеры адаптивности

```html
<!-- Текст размер меняется -->
<h1 class="text-2xl sm:text-3xl lg:text-4xl">Заголовок</h1>

<!-- Сетка меняется -->
<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
  <!-- 1 колонка на мобильных, 2 на планшетах, 3 на десктопе -->
</div>

<!-- Паддинг меняется -->
<div class="px-4 sm:px-6 lg:px-8">
  <!-- Разные отступы слева/справа -->
</div>
```

### Chat Message компонент

```tsx
interface MessageProps {
  role: 'user' | 'assistant';
  content: string;
}

export default function Message({ role, content }: MessageProps) {
  const isUser = role === 'user';
  
  return (
    <div className={`flex ${isUser ? 'justify-end' : 'justify-start'} mb-4`}>
      <div
        className={`max-w-md px-4 py-3 rounded-lg ${
          isUser
            ? 'bg-black text-white'
            : 'bg-gray-100 text-gray-900 border border-gray-200'
        }`}
      >
        <p className="text-sm leading-relaxed">{content}</p>
      </div>
    </div>
  );
}
```
