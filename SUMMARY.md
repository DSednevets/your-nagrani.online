## 🎯 ПРОЕКТ В ДВУХ СЛОВАХ

**your-nagrani.online** — это SaaS платформа с AI-ассистентом для глубокого исследования личности.

**Основная функция:** AI проводит диалог с пользователем, собирает "живой материал" и помогает человеку увидеть себя глубже.

**Целевая аудитория:** Мужчины 25-50 лет с выгоранием, неудовлетворенностью, желанием реализоваться.

**Модель:** Free trial (1 блок ИССЛЕДОВАНИЯ, ~15 вопросов) → Premium подписка ($29/мес)

## 🤖 AI ЛОГИКА (4 РЕЖИМА)

### 1. ИССЛЕДОВАНИЕ (начало, мало данных)
- Формат: отражение → наблюдение → 1 вопрос
- Цель: собрать материал, найти паттерны
- Пример: "Ты описал состояние где отпускаешь контроль. Похоже контроль ключевой механизм. Что ты пытаешься контролировать?"

### 2. СБОРКА (после 7-10 вопросов)
- Формат: структурирование → выводы → паттерны
- Цель: показать что повторяется
- Пример: "Смотри что повторяется: ты создаешь но не проявляешь, двигаешься через напряжение, обесцениваешь результат. Это одна система."

### 3. ПРОРЫВ (уход в оправдания)
- Формат: коротко, жестче, вскрытие
- Цель: разбить защиту, показать избегание
- Пример: "Ты сейчас не запутался. Ты избегаешь. Ты знаешь что делать но не делаешь."

### 4. ДЕЙСТВИЕ (есть ясность)
- Формат: конкретика, 1 шаг, без философии
- Цель: перевести в действие
- Пример: "Ок. Твой следующий шаг: довести НАГРАНИ до версии 1.0. Назначь дату. Что ты сделаешь завтра?"

## 🔄 ОСНОВНОЙ ФЛОУ

### Новый пользователь
```
1. Видит лендинг
2. Нажимает "Начать исследование"
3. Регистрация (email + password)
4. Автоматический вход
5. Открывается чат с первым вопросом:
   "Расскажи о моменте в жизни когда ты чувствовал: вот сейчас я настоящий"
6. Пользователь пишет ответ
7. ИИ отвечает (режим ИССЛЕДОВАНИЕ)
8. Этот цикл повторяется ~15 раз
9. На 15-м ответе пользователя:
   - ИИ дает обычный ответ
   - Затем добавляет ВЫВОД (3 линии + гипотеза)
10. Экран: "Требуется подписка для продолжения"
11. Кнопка "Оформить подписку" → Stripe checkout
12. Оплата
13. Redirect на /dashboard
```

### Возвращающийся пользователь
```
1. Логин
2. Dashboard (список всех своих диалогов)
3. Нажимает "Продолжить" на каком-то диалоге
4. Загружается вся история этого диалога
5. Открывается чат
6. ИИ переключается в соответствующий режим
7. Диалог продолжается
```

## 🔑 КЛЮЧЕВЫЕ МОМЕНТЫ

### 1. Счетчик вопросов
- message_count начинается с 0
- После 15-го ответа пользователя (message_count === 14) → вывод
- После вывода → check подписки
- Если подписки нет → redirect на /pricing

### 2. Системный промт (в API)
- Полный промт находится в `AI_ASSISTANT_SYSTEM_PROMPT.md`
- Копируй его целиком в `/api/ai/process-message/route.ts`
- Он определяет весь стиль и логику ассистента

### 3. История диалогов
- Все сообщения сохраняются в БД (таблица `messages`)
- При загрузке диалога вся история передается в Claude API
- message_count должен быть правильным (количество ответов ассистента)

### 4. Подписка
- Free блок: первые 15 вопросов (FREE_TRIAL режим)
- После 15 → check подписки в API
- Если нет подписки → 403 Forbidden
- После оплаты → subscription.status = 'active'

### 5. Дизайн
- Все компоненты в Tailwind
- Используй классы из Design Guide
- Светлая тема (#FFFFFF фон, #000000 текст)
- Responsive на всех размерах экранов

```
POST /api/ai/process-message
Input: conversation_id, user_message, conversation_history, message_count
```

```
GET /api/chat/history/[conversation_id]
GET /api/chat/conversations
```

```
GET /api/subscription/check-status
POST /api/subscription/webhook (Stripe)
```

```
(не нужно создавать свои endpoints)
```
